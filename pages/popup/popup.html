You will need to play the game with the devtools "KanColle" panel opened. Activate the devtools panel with F12 or Ctrl+Shift+J. With the devtools open, find and select the "Kancolle" tab from the top menu of the devtools. You may need to press the ">>" in order to find it. Remember to NOT let the game start until this panel has been activated.
